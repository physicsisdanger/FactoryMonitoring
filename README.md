# Factory Monitoring System

Система мониторинга состояния станков в реальном времени.

## Технологии

- **Frontend**: React, Vite, Zustand, react-grid-layout
- **Backend**: FastAPI (в разработке)
- **Database**: PostgreSQL, InfluxDB (в разработке)

## Установка и запуск

### Frontend

```bash
cd frontend
npm install
npm run dev
```
